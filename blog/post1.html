OSINT con imágenes y análisis por IA

Publicado el 18 de junio de 2025
Por Iago Lorenzo Manrique

Código fuente disponible en: github.com/IagoLorenzo/OSINT-Images-IA

En esta entrada comparto un proyecto que combina OSINT (Open Source Intelligence) con el análisis automatizado de imágenes públicas procesadas con la IA de Google (gemini-2.0-flash) para extraer información útil.

Este proyecto es solo para fines educativos, o ejercicios de formación en inteligencia. No debe usarse para espiar o vigilar a personas reales sin consentimiento explícito.

Objetivo

Automatizar el análisis de imágenes para generar un informe OSINT detallado, utilizando inteligencia artificial.

¿Que es el OSINT?

El OSINT (Open Source Intelilligence) es el proceso de recolectar, analizar y utilizar información pública disponible para obtener datos útiles para un objetivo. No se refiere solo a lo que hay en Google, sino a absolutamente todo lo que se pueda recolectar sin requerimiento de métodos intrusivos.

Herramientas utilizadas

  • Python con módulos como PIL, requests y dotenv
  • Google AI Studio utilizando el modelo gemini-2.0-flash

Estructura del proyecto

Puedes consultar la estructura detallada del proyecto y todos los archivos de código en el repositorio oficial:

github.com/IagoLorenzo/OSINT-Images-IA

Funcionamiento general

El programa analiza automáticamente cada imagen y genera un informe consolidado con información útil. Utiliza una API de Google para realizar el análisis semántico y visual.

Resultado

Como resultado, se obtiene un informe detallado y profesional redactado por la IA en formato .pdf convertido desde markdown, que resume lo más relevante de todas las imágenes analizadas, incluyendo posibles ubicaciones, objetos visibles, texto en carteles, personas, símbolos y cualquier dato potencialmente útil desde el punto de vista OSINT.
